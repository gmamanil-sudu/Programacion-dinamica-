# Programación Dinámica

Este repositorio contiene la solución de dos problemas clásicos utilizando Programación Dinámica en C++.

## Problemas resueltos

### 1. Escaleras

Se calcula la cantidad de formas diferentes de llegar al escalón `N`, permitiendo avanzar únicamente 1 o 2 escalones. La solución utiliza Programación Dinámica con memoización para evitar cálculos repetidos.

### 2. Cambio mínimo de monedas

Dado un conjunto de monedas y una cantidad objetivo, se determina la cantidad mínima de monedas necesarias para formar dicha cantidad. Además, se muestra una combinación válida de monedas y la tabla DP generada.

## Tecnologías utilizadas

* C++
* Programación Dinámica
